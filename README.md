## 🧑🏻‍💻 Installation

Check Node.js installation and version by running the command in the terminal

```
$ node -v
```

Clone the project to local

```
$ git clone https://github.com/LJBL22/react_twitter.git
```

Go to the directory

```
$ cd react_twitter
```

Install npm

```
$ npm install
```

Start the project

```
$ npm start
```
